CORS 오류 — localhost:5173 → :8080 교차 출처 차단
원인: React 개발 서버(5173)에서 Spring(8080)으로의 cross-origin 요청 차단

파일 서빙 안됨 — C:/FFG/upload 로컬 경로 브라우저 접근 불가
원인: 로컬 파일 시스템 경로를 브라우저가 직접 접근 불가
WebMvcConfigurer /uploads/** → file:///C:/FFG/upload/ 정적 매핑
SHAP JSON 직렬화 오류 — NumPy float32
원인: NumPy float32 타입은 Python 기본 json.dumps 직렬화 불가
round(float(val), 4)로 Python 기본 float 변환 후 jsonify() 반환
Oracle ROWNUM 페이징 정렬 오류
원인: ORDER BY 전에 ROWNUM 적용 → 정렬 후 페이징이 아닌 페이징 후 정렬 발생
서브쿼리에서 ORDER BY 먼저, 외부 쿼리에서 ROWNUM 필터 적용
